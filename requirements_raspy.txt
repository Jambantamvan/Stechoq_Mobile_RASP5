# ===== INMP441 I2S MICROPHONE NOTES =====
# Untuk menggunakan INMP441, tambahkan ke /boot/config.txt:
#   dtparam=i2s=on
#   dtoverlay=googlevoicehat-soundcard
# atau:
#   dtoverlay=i2s-mems-mic
# Lalu reboot Raspberry Pi
#
# Wiring INMP441 ke Raspberry Pi 5:
#   VDD  -> 3.3V (Pin 1)
#   GND  -> GND (Pin 6)
#   SD   -> GPIO 21 (Pin 40) - I2S Data
#   WS   -> GPIO 19 (Pin 35) - I2S Word Select (LRCLK)
#   SCK  -> GPIO 18 (Pin 12) - I2S Clock (BCLK)
#   L/R  -> GND (Left channel) atau 3.3V (Right channel)
